# n8n-Generate-RFP-proposal-drafts-from-Google-Drive-with-Groq-AI-Salesforce-and-Slack
AI-powered RFP Automation workflow built with n8n, Google Drive, Groq AI (LLaMA 3), Salesforce, Google Docs, and Slack. Automatically extracts RFP requirements, generates proposal drafts, creates CRM opportunities, assigns review tasks, and streamlines proposal management.
